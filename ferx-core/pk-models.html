Analytical PK Models

ferx-core provides closed-form analytical solutions for one-, two-, and three-compartment PK models. These are the preferred structural model type: they are faster than ODE integration and support the Enzyme automatic differentiation path for gradient computation.

Invoking a model in the DSL

The [structural_model] section of a .ferx file selects an analytical model with the pk keyword:

[structural_model]
  pk one_cpt_oral(cl=CL, v=V, ka=KA)

Named arguments map individual-parameter symbols to the model’s PK slots. The available models and their required arguments are listed below.

One-compartment models

All one-compartment solutions are in pk/one_compartment.rs.

IV bolus

pk one_cpt_iv_bolus(cl=CL, v=V)

\[C(t) = \frac{\text{Dose}}{V} \cdot e^{-k \cdot t}, \quad k = \frac{CL}{V}\]

IV infusion

pk one_cpt_iv_infusion(cl=CL, v=V)

During infusion (\(t \le T_{\text{inf}}\)): \[C(t) = \frac{\text{Rate}}{CL} \cdot (1 - e^{-k t})\]

After infusion (\(t > T_{\text{inf}}\)): \[C(t) = \frac{\text{Rate}}{CL} \cdot (1 - e^{-k T_{\text{inf}}}) \cdot e^{-k(t - T_{\text{inf}})}\]

Oral absorption

pk one_cpt_oral(cl=CL, v=V, ka=KA)
pk one_cpt_oral_f(cl=CL, v=V, ka=KA, f=F)   # with bioavailability

\[C(t) = \frac{F \cdot \text{Dose} \cdot K_A}{V (K_A - k)} \left(e^{-k t} - e^{-K_A t}\right)\]

When \(K_A \approx k\) (flip-flop, within 1e-6), the L’Hôpital limit is used to avoid catastrophic cancellation:

\[C(t) = \frac{F \cdot \text{Dose} \cdot K_A}{V} \cdot t \cdot e^{-k t}\]

Two-compartment models

All two-compartment solutions are in pk/two_compartment.rs.

Macro rate constants are derived from micro-constants using Vieta’s formula to avoid subtracting two nearly-equal large numbers when \(s \gg \sqrt{s^2 - 4d}\):

let alpha = (s + disc) / 2.0;
let beta  = d / alpha;           // Vieta: α·β = d

where \(s = k_{10} + k_{12} + k_{21}\) and \(d = k_{10} \cdot k_{21}\).

IV bolus

pk two_cpt_iv_bolus(cl=CL, v1=V1, q=Q, v2=V2)

\[C(t) = A e^{-\alpha t} + B e^{-\beta t}\]

\[A = \frac{\text{Dose}}{V_1} \cdot \frac{\alpha - k_{21}}{\alpha - \beta}, \quad B = \frac{\text{Dose}}{V_1} \cdot \frac{k_{21} - \beta}{\alpha - \beta}\]

IV infusion

pk two_cpt_iv_infusion(cl=CL, v1=V1, q=Q, v2=V2)

Superposition of the bolus solution across the infusion duration; switches to bolus formula when duration ≤ 0.

Oral absorption

pk two_cpt_oral(cl=CL, v1=V1, q=Q, v2=V2, ka=KA)
pk two_cpt_oral_f(cl=CL, v1=V1, q=Q, v2=V2, ka=KA, f=F)

\[C(t) = Ae^{-\alpha t} + Be^{-\beta t} + Ce^{-K_A t}\]

Coefficients derived analytically; degenerate cases (\(K_A \approx \alpha\) or \(K_A \approx \beta\)) handled via L’Hôpital limits.

Three-compartment models

Three-compartment solutions are in pk/three_compartment.rs. Macro rate constants (\(\alpha\), \(\beta\), \(\gamma\)) are solved from the characteristic polynomial using Cardano’s formula.

pk three_cpt_iv_bolus(cl=CL, v1=V1, q2=Q2, v2=V2, q3=Q3, v3=V3)
pk three_cpt_iv_infusion(cl=CL, v1=V1, q2=Q2, v2=V2, q3=Q3, v3=V3)
pk three_cpt_oral(cl=CL, v1=V1, q2=Q2, v2=V2, q3=Q3, v3=V3, ka=KA)

Multi-dose superposition

All analytical models support multiple dose events. The pk/mod.rs dispatcher calls the single-dose solution for each historical dose and sums the contributions at each observation time:

// Conceptual superposition
for dose in &subject.doses {
    let t_since_dose = t_obs - dose.time;
    if t_since_dose >= 0.0 {
        pred += single_dose_solution(dose, t_since_dose, &pk_params);
    }
}

PkParams layout

PK parameters are passed through a fixed-layout [f64; 8] array rather than a HashMap so that Enzyme’s reverse-mode AD can differentiate through them:

Index Symbol Description
0 CL Clearance
1 V / V1 Volume (central)
2 Q Intercompartmental clearance (central ↔︎ peripheral 1)
3 V2 Peripheral volume 1
4 KA Oral absorption rate constant
5 F Bioavailability (default 1.0)
6 Q3 Intercompartmental clearance (central ↔︎ peripheral 2; 3-cmt)
7 V3 Peripheral volume 2 (3-cmt)

Unused slots default to 0.0 (except F, which defaults to 1.0).
